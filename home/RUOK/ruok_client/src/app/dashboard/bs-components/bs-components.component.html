Bootstrap components for Angular

A demonstration of the components available in ng2-bootstrap from valour-software

Each component is a minimal sample demonstration only. Whilst I endevour to keep these in sync with the latest versions from valor software the user is strongly encouraged to follow each demos link. This will take you to the relevant ng2-bootstrap page where the full demo, information and usage instructions are provided.

Note: Bootstrap 4 does not include glyphicons anymore, so if you want to use that font, you will need to add a link to glyphicons.css. This package has been built using Font Awesome and all the icons changed to the nearest Font Awesome equivelent.
